# FlowBatch landing (Vite + React + Tailwind)

Switched from Next.js to **Vite** so `npm install` fits on tight disks.

```bash
cd web
npm install
npm run dev
```

Open **http://127.0.0.1:5173/**

- Entry: `src/main.jsx`
- Page: `src/LandingPage.jsx` (copy of `landing-page_1.jsx`)

Extension: load unpacked from **`../extension/`**.
